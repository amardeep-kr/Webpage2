16. Form Action

The action attribute tells the form where to send data when submitted.

<form action="submit_page.html" method="POST">

17. Submit Method (GET vs POST)

GET: Data is sent in the URL (visible). Good for search forms.


POST: Data is sent in the request body (hidden). Good for login/signup forms.

18. Setting Up HTML Locally

To set up HTML locally you need:

  1. A code editor like VS Code — Download here
  2. Install the Live Server extension in VS Code
  3. Create a new file with .html extension
  4. Right-click the file → Open with Live Server
